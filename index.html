تنظیمات IPV6 ایرانسل

برای اعمال تنظیمات لطفا پروفایل زیر رو دانلود کرده و آموزش های زیر رو دنبال کنید

  • روی پروفایل کلیک کنید و بعد ALLOW را بزنید.
  • به بخش Settings > Profile Downloaded رفته و پروفایل را نصب کنید.
  • به Settings > Cellular بروید و بر روی Cellular Data Network کلیک کنید.
  • از قسمت SIMs، سیمکارت ایرانسل را انتخاب کنید.
  • به Cellular Data Network رفته و به پایین صفحه اسکرول کنید.
  • برای تنظیمات شبکه IPV6، Reset Settings را بزنید.
